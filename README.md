### Prerequisites
- Node.js v18+
- Angular CLI v13+
- MongoDB Compass (optional)

### Setup Instructions

1. **Clone repository**:
   ```bash
   git clone [your-repo-url]
   cd Moodle-Angular
   

2. **Backend Setup**:
   ```bash
    cd backend
    npm install

3. **Frontend Setup**:
    ```bash
    cd Moodle-Angular 
    npm install

4. **Running the website**:
    - In one terminal (backend):
      ```bash
        cd backend
        npm start
    - In another terminal (frontend)
      ```bash
        cd Moodle-Angular
        ng serve
